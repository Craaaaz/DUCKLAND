--------------------------------------------------------------------------------
遊戲參考: 文明帝國系列以及世紀帝國系列

遊戲名字: DUCKLAND

遊戲玩法: 4X回合制文明遊戲

遊玩人數: 2-4

元素: 卡牌 經營 PVP 棋盤類地形(4X)

卡牌類型: 建築類 強化類 研究類 經營類 兵種類

文明: 擅長魔法、擅長近戰、擅長經營

地圖造型: 2人(長方形)、3人(六邊形)、4人(六邊形)

--------------------------------------------------------------------------------
開局: 國王(卡片持有者)

中場: 將軍(卡片代理)

勝利條件: 國王死亡 城堡破壞 奇觀勝利

--------------------------------------------------------------------------------
資源: 黃金 知識點

--------------------------------------------------------------------------------
基礎卡牌設計:

建築類卡牌: 城堡牌(開局自帶，核心和購買卡牌)、學校(生產知識點、解鎖研究類卡牌)、商行(賺錢、跟小型村莊貿易)、醫院(幫兵種回血) 兵工廠

強化類卡牌: 圖書館(增加學校產生效率) 馬車(增加商行產生效率) 武館(增加產生兵種初始數值)

研究類卡牌: 文字記錄(可建造圖書館) 訓馬場(可建造馬車)

經營類卡牌: 貿易路線、鼓舞民眾(提高建築效率)

兵種類卡牌: 將軍 戰士 法師 工程器 工程人員

建築類以及兵種類卡牌具備數值: 血量(整數) 基礎攻擊(整數) 速度(0-10) 回復效率(百分比)

--------------------------------------------------------------------------------
邏輯/

A. 場地建立邏輯:
1. 每個單位的場地皆為6邊形同尺寸大小
2. 場地中每場會隨機生成地圖資源點 數量可經由開始前房主調整(地下城(有怪物)、小型村莊(可交易)、野外寶藏(經過即可獲取隨機物資))
   條件: 不生成在玩家出生點4格內 同時資源點3格內不會有兩個以上資源點

B. 卡片建立邏輯:
1. card資料夾中會有關於初始卡片的設定 如:卡片樣式(形狀 大小...等)
2. 建立新卡牌時需輸入(按系統現有內容作勾選): 等級 獲得條件 作用範圍 功能:如增加射程10% ...(後續補充) 以及卡片名稱及說明文字: 如名稱: 基礎劍術 說明: 可作用於近戰兵 增加選定兵種10%基礎攻擊力
3. 完成後卡牌會以json或其餘資料儲存的方式 以資料的方式儲存 而非新增一張卡牌就需要新增程式的方式

--------------------------------------------------------------------------------
系統邏輯:
當按下開始按鈕後
首先建立地圖
以及將玩家操控之國王放入地圖的兩側 三以及四名為家則是在六個角隨機選擇不重覆點位

遊戲開始後 玩家可操控國王進行移動(基礎行動力為5點) 以及放置卡牌 (基礎行動力為5點 移動以及放置卡牌皆消耗1點) 完成後可按下結束回合 當所有玩家皆按下結束回合後

則結束當前回合 並開始計算所有玩家的操控產生的影響(包括角色位置移動 建築產生物資 戰鬥造成影響)

由於所有玩家屬於同步回合 所以會戰鬥時會出現雙方同時攻擊對方之行為 則透過受影響之卡牌速度數值影響

補充(問答整理):

流程與設定:
- 房間建立: 地圖大小(固定三張地圖供選擇)、資源點數量可設定
- 開始前: 選擇文明(魔法/近戰/經營)，影響初始資源、國王數值、後續研究與強化卡牌
- 第1回合: 國王擁有一張城堡卡，可放置城堡；城堡可購買卡片並作為核心
- 每回合流程: 行動所有可行動物件(卡牌使用、角色移動等) → 結束回合 → 結算階段
- 特殊事件: 目前無

卡牌放置:
- 卡牌可放置在持有卡牌者周遭3格內
- 建築放置後即存在，國王離開後仍保留
- 可建立在敵方領土
- 建築相鄰會有額外增益效果(細節後續補充)

戰鬥系統:
- 由玩家主動觸發
- 雙方皆存活時必定可攻擊
- 傷害公式: 基礎攻擊力 * (1 + 額外加成)
- 無防禦力或減傷
- 戰鬥於回合結束時結算
- 若原定觸發攻擊但在結算時先被擊殺，則攻擊失效
- 攻擊速度相同時，採隨機優先制

多人與勝利:
- 目前無AI或單人模式，優先實現網路功能
- 勝利條件: 國王死亡、城堡破壞、奇觀勝利(奇觀=特定建築完成)

資源與經濟:
- 產出來源: 建築效果、貿易流通、擊殺生物、佔領村莊
- 回合結算順序: 計算所有生產 → 扣除回合損失(維持費) → 顯示結果
- 所有建築皆有維持費
- 無資源上限

卡牌取得與稀有度:
- 主要取得: 城堡購買
- 分類: 基礎卡、解鎖後可購買卡、各文明特色卡
- 後期建築: 交易所可交易獲得原價9折
- 無卡牌上限
- 有稀有度區分

單位行動與視野:
- 行動力: 國王5、將軍6，其餘兵種待設計
- 阻擋: 敵方建築會阻擋移動
- 視野: 需有己方兵種或建築才可獲得視野
- 無視野區域無法看到其上方是否有兵種或建築

奇觀規則:
- 以高額黃金建立
- 若特定回合內奇觀未被破壞，達成奇觀勝利
- 全地圖同時僅可存在一座奇觀

相鄰增益(建築):
- 觸發範圍: 相鄰1格
- 需特定組合(組合清單後續補充)
- 增益為百分比方式(可疊加規則後續補充)

單位數值(已定義):
- 國王: 血量1000、攻擊50、速度7、射程1、行動力5、視野3
- 國王可隨達成任務進行升級(升級內容後續補充)
- 其餘兵種數值待設計（草案）
    將軍：血量 600 / 攻擊 40 / 速度 6 / 射程 1 / 行動力 6 / 視野 3
    戰士：血量 400 / 攻擊 35 / 速度 5 / 射程 1 / 行動力 4 / 視野 2
    法師：血量 250 / 攻擊 50 / 速度 5 / 射程 3 / 行動力 2 / 視野 3
    工程器：血量 220 / 攻擊 20 / 速度 4 / 射程 1 / 行動力 4 / 視野 2
    工程人員：血量 180 / 攻擊 10 / 速度 4 / 射程 1 / 行動力 6 / 視野 2

回合結算順序:
- 移動結果 → 戰鬥判定 → 死亡處理 → 資源生產 → 維持費 → 勝利判定

回合限制:
- 目前無回合上限或計分結算
